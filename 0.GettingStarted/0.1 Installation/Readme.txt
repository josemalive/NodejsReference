Steps

1º go to http://nodejs.org/
2º Install node in your machine using the last released version
3º After installation open Nodejs command prompt and write "node -v" this will render the version installed so you can verify that's properly installed.

Pasos

1º Ir a http://nodejs.org
2º Descargar la ultima version publicada de nodejs e instalarla.
3º Tras la instalación, abre el prompt de nodejs y escribe "node -v", esto mostrará la versión instalada en tu máquina.
