Install Steps:

1. Go to http://nodejs.org/
2. Install node in your machine using the last stable version
3. After installation open Nodejs command prompt and write "node -v" this will render the version installed so you can verify that's properly installed.

Instalación:

1. Ir a http://nodejs.org
2. Descargar la version mas estable publicada de nodejs e instalarla.
3. Tras la instalación, abre el prompt de nodejs y escribe "node -v", esto mostrará la versión instalada en tu máquina.
